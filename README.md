# Simple Streamlit Graphs App

A simple Streamlit application that displays various types of graphs.

## Features

- Line Chart
- Bar Chart
- Scatter Plot
- Histogram

## Installation

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit app:
```bash
streamlit run app.py
```

The app will open in your default web browser, typically at `http://localhost:8501`.

## Graph Types

- **Line Chart**: Displays sine and cosine functions with sample data
- **Bar Chart**: Shows a colorful bar chart with random values
- **Scatter Plot**: Visualizes random data points with color mapping
- **Histogram**: Displays the distribution of random normal data

Use the sidebar to switch between different graph types.
